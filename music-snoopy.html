MoodMix AI

A vibrant, mood-driven playlist generator using AI to turn any emotion or vibe into a curated soundtrack.

MoodMix is an experimental AI DJ concept I designed and built, imagining a personal music-curation engine. Users type in a mood → “sunny afternoon roadtrip,” “lo-fi laundry night,” “foggy 6AM freeway” — and the model generates a color palette and playlist that match the emotional tone.

The system uses unsupervised music embeddings to understand sonic similarity and supervised NLP to interpret mood prompts. Combined with Spotify’s audio features (energy, valence, tempo), the tool recommends songs that genuinely feel like the moment you're describing.

  • Color-driven UI inspired by vinyl, pins & Snoopy aesthetics
  • Gradient-based mood palettes generated from embeddings
  • Custom playlist cards (Sunshine Drive, Easy Going, Cruisin’ Tunes)
  • AI-interpreted text prompts for instant mood recognition
  • Prototype built with HTML, CSS, JavaScript, and Spotify Web API

This project blends design, frontend engineering, and playful machine-learning concepts, resulting in a warm, joyful interface that feels like a friend making you a playlist.
