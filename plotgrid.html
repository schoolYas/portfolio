PlotGrid – Visual Plot Mapping for Writers

A colorful web tool for mapping characters, events, conflicts, and settings on a shared grid, helping writers see their stories as connected systems instead of scattered notes.

PlotGrid started as a thought experiment: what if outlining a novel felt more like arranging a constellation than editing a bulleted list? I designed and built a prototype interface where writers drag nodes representing characters, events, conflicts, and settings onto a shared grid, then connect them with visible relationship lines.

The UI is inspired by sticky notes, index cards, and crime-board diagrams, but reimagined for the browser. Each node is color-coded by type and placed on a soft grid, while connectors reveal how plot threads intersect and depend on each other.

  • Visual grid of story elements (characters, events, conflicts, settings)
  • Color-coded nodes with short, readable titles for each plot point
  • Connector lines that make relationships between scenes and characters explicit
  • Concept support for AI-powered critique of structure, pacing, and stakes
  • Built with semantic HTML, modern CSS, and a focus on clean, writer-friendly UX

PlotGrid is designed for novelists, game writers, and storytellers who think spatially. Instead of hiding structure inside documents, it pulls everything onto one canvas to surface patterns, gaps, and overloaded sections at a glance.
